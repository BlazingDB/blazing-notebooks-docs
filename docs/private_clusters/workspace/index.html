Jupyter Lab extensions¶

Dask JupyterLab extension¶

The Dask community develops and supports a JupyterLab extension that allows Dask dashboard charts to be into a JupyterLab session. This allows the user to view diagnostic diagrams in an easy way thus generating a better user experience.

The Dask JupyterLab extension is already installed for private clusters created in the BlazingSQLNotebooks app and when you lauch JupyterLab there you will see a tab with Dask logo in the left sidebar.


Then you can select the best option for you and construct your ideal workspace.
