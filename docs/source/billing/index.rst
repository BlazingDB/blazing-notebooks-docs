=======
Billing
=======

BlazingSQL Notebooks offers you a friendly interface to buy 
credits and start working on private clusters. 

Buying credits
==============

In each pages of BlazingSQL Notebooks app, you will have the followed section 
and there you will see your available number of credits.

.. raw:: html

    <img src="../_static/images/init.png" alt="init"/>

|

Purchasing credits required to complety only two steps: 

First, you have to select how many credits you want to buy. 

.. raw:: html

    <img src="../_static/images/buying_step_1.png" alt="select_credits" />

|

Then, you will have to enter you credit card information and do click on 
*"Pay $ ..."* button. 

.. raw:: html

    <img src="../_static/images/buying_step_2.png" alt="pay_for_credits" />

|

Finally, you will see a confirmation message.

.. raw:: html

    <img src="../_static/images/buying_step_3.png" alt="confirmation_message" />

|

Aditionally, in the *"Total credits"* section you will see the 
the number of available credits were updated.

.. raw:: html

    <img src="../_static/images/end.png" alt="end" />



Tracking spending
=================

When you are logged into BlazingSQL Notebooks apps you will the
*"credits"* menu in the sidebar.

.. raw:: html

    <img src="../_static/images/billing_menu.png" alt="credits" />

|

When you click on the *"credits"* menu, the system will show you there
graphics.

The first one is about the used credits by day in the select month. By 
default will be the current month.

.. raw:: html

    <img src="../_static/images/graphic_1.png" alt="end" />

|

The second graphic will show you information related to the used credits
used by session in the selected month.


.. raw:: html

    <img src="../_static/images/graphic_2.png" alt="end" />

|

Finally, in the last graphic you will see the total used credits and 
the time used (in minutes) per day.

.. raw:: html

    <img src="../_static/images/graphic_3.png" alt="end" />



Pricing
=======

Each credit cost $1 and the cost per hour and GPU is 1 credit.

.. list-table:: Pricing per hour and number of GPUs
   :widths: 25 25 25
   :header-rows: 1

   * - SKU
     - Number of GPUs
     - Cost per hour
   * - **X-Small**
     - 1
     - $1
   * - **Small**
     - 2
     - $2 
   * - **Medium**
     - 4
     - $4 
   * - **Large**
     - 8
     - $8 
   * - **X-Large**
     - 16
     - $16 
   * - **2X-Large**
     - 32
     - $32 
   * - **3X-Large**
     - 64
     - $64
