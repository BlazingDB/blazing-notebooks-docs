Billing¶

BlazingSQL Notebooks offers you a simple and friendly interface to buy credits so you can start working on private clusters with ease.

Credits¶

Credits are the currency of Blazing Notebooks. By purchasing credits you gain access to the easiest way to launch GPU clusters with up to 64 GPUs (see private clusters).

Available credits¶

On the top of the BlazingSQL Notebooks app page, a current total number of remaining credits is always visible. As long as you have some credits available you can create and use private clusters on Blazing Notebooks.


Purchasing credits¶

Purchasing credits requires only two steps:

  1. Select how many credits you want to buy.


  1. Enter you credit card information and click on Pay button.


Once the transaction is verified you will see the confirmation message.


Having successfully finalize the transaction, the Available credits will be updated with the additional amount you just purchased.


Tracking spending¶

Blazing Notebooks provide provide capabilities for you to track your spending. Once logged into Blazing Notebooks select the credits from the sidebar.


This first view shows the usage per day in a selected month (December in the example below). The view defaults to the current month.


The second view shows the information related to the credits used per session in the selected month. The view defaults to the current month.


Finally, in the last view shows the total used credits and the time used (in minutes) per day.
